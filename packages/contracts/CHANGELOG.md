### Minor Changes

- Initial release of observability contracts package
  - Provider-agnostic interfaces for metrics, tracing, and logging
  - IMetricsProvider, ITracingProvider, ILoggingProvider contracts
  - ObservabilityConfig type definitions
  - Comprehensive documentation

  Enables pluggable observability providers without coupling to specific implementations.
