## Installation

```bash
# Core packages (automatically installed)
npm install @gati-framework/runtime
npm install @gati-framework/cli

# Cloud provider plugins (optional)
npm install @gati-framework/cloud-aws

# Development tools (optional)
npm install @gati-framework/playground
```

## Usage Examples

### AWS Deployment

```typescript
import { AWSCloudProvider } from '@gati-framework/cloud-aws';

const provider = new AWSCloudProvider();
await provider.initialize({
  region: 'us-east-1',
});

await provider.createCluster({
  name: 'my-gati-cluster',
  version: '1.28',
  nodePools: [{
    name: 'default',
    instanceType: 't3.medium',
    minNodes: 2,
    maxNodes: 10,
  }],
});
```

### Playground Integration

```typescript
import { createApp } from '@gati-framework/runtime';
import { playgroundModule } from '@gati-framework/playground';

const app = createApp();

// Add playground module for debugging
app.use(playgroundModule({
  enabled: process.env.NODE_ENV === 'development',
  port: 3001,
}));
```
